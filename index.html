EESE Wrike Process

The following are steps to crete a timeline within Wrike


Option 1) Where to keep the new schedule

In this example a folder is created within EESE Project Schedule Holding.
After creating the folder, copy the name.
View the process for placing in an existing schedule


Locate where the blueprints are

The blue print folder is located under the hamburger menu near the bottom of the project listing


Create from Blueprint

Right click on the desired blueprint and choose "Create from Blueprint"


Choose a name for the new timeline

This will be the primary name used to identify the timeline


Choose a location for the new timeline

The location can either be within an existing project or in the folder named "EESE Production Schedule Holding"


Reschedule

Choose "due date" and select the desired drop or blast date


Create and locate the new schedule

Click on create to create the new time line with the desired date, in the requested location, then locate it


Assigning task owners (mass assignment)

To assign an individual contributor to more than one task,
choose the schedule,
expand task list to view tasks that are to be assigned
hold the shift key and select the first task to be assigned (a check mark will appear next to the task chosen)
choose additional tasks to be assigned to the same individual
using the menu on the left choose add assignee and pick the individual responsible for those tasks


Checking Dates: Introduction to the Gnatt Chart

For viewing and adjusting dates, choose the Gnatt Chart view
(often things will appear out of order, to order correctly click on "Title")
When you open the Gnatt Chart view, it automatically finds today, (indicated by a red vertical line),
if the schedule you are working on is a week or two away, you will not see the flow, to see the flow, click the gray arrow on the left side of the Gnat Chart view.
To expand the view so things are easier to work with, click the expand button (looks like a box inside a box)
To adjust the schedule, you can either move the boxes or enter a new date. All dates after the date adjusted will shift to reflect current task time requirements
(Wrike automatically skips holidays)
The Blast Date, known as the "Deployment Drop" in wrike, will not move automatically.
The new expected deployment drop date will need to be entered manually.


Troubleshooting: Tasks are out of order

Help! My tasks are displaying out of order!
Reason: "Sort by" is not set to "by title"
Remedy: change "Sort by" to "by title"


Troubleshooting: Tasks are missing

Question: Where did all the tasks go, I'm only seeing one or two tasks showing!
Reason: View is filtered, showing only "active tasks"
Remedy: Change view to "All tasks"


Placing a schedule in an existing project

To add to an existing project, find the project within wrike using a portion of the project name to locate it.
Copy the project name
Go to the blueprints and choose the schedule you wish to include
After naming the schedule, past the copied project name into the "select folder" box and choose the project from the dropdown.
After completing the rest of the steps to create a schedule from a blueprint, the schedule will appear at either the bottom or the top of the project task list depending upon how the project is sorted.
Drag the schedule into the number 60 eese task.
